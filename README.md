# AI Productivity Dashboard

An intelligent task management system that uses AI to help you get more done. It learns from your work patterns and suggests the best times to tackle different tasks.

## What it does

This dashboard helps you organize your tasks, prioritize what matters, and get personalized suggestions on how to be more productive. You can drag and drop tasks, set priorities, and track your progress over time. The AI component analyzes your work patterns and gives you insights on when you're most productive.

## Built with

- **Frontend:** React.js with drag-and-drop functionality
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **AI:** Machine learning models for productivity insights

## Main features

- Interactive task board with drag-and-drop
- Priority management system
- AI-powered task suggestions
- Productivity analytics and insights
- Real-time task tracking
- Personalized recommendations based on your work habits

## The idea behind it

I built this because I was struggling to manage multiple projects and deadlines during college. I wanted something that not only helped me organize tasks but also understood when I work best and suggested schedules accordingly. The result is a tool that's helped me improve my task completion rate significantly.

## Getting started

1. Clone this repository
2. Install dependencies for both client and server
3. Set up your MongoDB database
4. Configure environment variables
5. Start the development servers

The AI features work better as you use the app more - it learns from your patterns over time.
